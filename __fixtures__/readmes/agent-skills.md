# agent-skills

Claude Code plugin marketplace for PatientVibes agent skills. Currently distributes the `pr-review-tools` plugin.

## Install

```bash
/plugin marketplace add D:/agent-skills
/plugin install pr-review-tools@patientvibes-skills
```

## Plugins

| Plugin | Status | Description |
|---|---|---|
| pr-review-tools | v1 | AI-driven PR review |

See [CONTRIBUTING.md](./CONTRIBUTING.md) for how to add a new plugin.
